About WECA: Mapped

This is a website is a Bath: Hacked project. Our goal is to help provide a resource that will allow local businesses, community groups and projects to find and explore data about the West of England Combined Authority (WECA).

Using this website you can explore data published by the three councils that make up the authority, as well as relevant extracts of national and international datasets, or data contributed by local community groups and volunteers.

This is not an official WECA project. We are a group of volunteers exploring how to better publish and use data for the benefit of the local area.

To see what data is available, click the Data Catalogue entry in the main map window.

How to use the Map

Go to main map page. Open the data catalogue and browse to find an entry. You can add any layer to the map.

The website is built on the same software as the Australian national map. You can read their help and how to documentation for more information.

The Software

This website is based on TerriaJS which was initially developed by Data61 for the Australian NationalMap, but has subsequently been used for other projects. It has been made available to assist with the development of other spatial data viewing applications.
